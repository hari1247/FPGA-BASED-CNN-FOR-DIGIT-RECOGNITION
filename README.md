# FPGA-BASED-CNN-FOR-DIGIT-RECOGNITION
Real-time digit recognition using a CNN on Zynq-7000 FPGA. Trained on MNIST with TensorFlow and deployed using INT8 quantization for efficient, low-power inference. A Python GUI captures input and sends it via UART for fast hardware-based prediction.
